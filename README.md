**metrics.** é um ecossistema SaaS completo para gestão pessoal e monitoramento de métricas essenciais. Desenvolvido com foco em uma experiência mobile-first premium, o Hub consolida rastreamento de hábitos, finanças, estudos e muito mais em um único arquivo portátil.

## 🚀 Funcionalidades Principais

- **Painel Master (CEO):** Gestão total de usuários, controle de precificação e portal de aprovação PIX.
- **Auto-Liberação PIX:** Sistema inteligente de ativação imediata mediante comprovante de pagamento.
- **Onboarding em Vídeo:** Tutorial dinâmico integrado via IndexedDB para instalação PWA.
- **Single File Distribution:** Compilação ultra-otimizada em um único HTML para portabilidade máxima.
- **Mobile-First Engine:** Interface de simulação de smartphone nativa para acesso Desktop e Mobile.

## 🛠️ Tecnologias Utilizadas

- **Core:** React 19 + Vite 8
- **Estilização:** Vanilla CSS (Glassmorphism & Sober Aesthetic)
- **Persistência:** LocalStorage + IndexedDB (para mídias pesadas)
- **Visualização:** Chart.js para dashboards de métricas
- **Ícones:** Lucide React

## 📦 Como Rodar o Projeto

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gere a build de arquivo único:**
   ```bash
   npm run build
   ```
   *O arquivo resultante será gerado em `dist/index.html`.*

## 📄 Estrutura de Matrículas (ID)
O sistema utiliza o padrão `X-YYYY` (Prefixo + Hash alfanumérico), garantindo IDs únicos e profissionais para seus clientes SaaS.
